 Medical Store Inventory & Reporting Tracker (Advanced Excel)

 📌 Project Overview

This project is an Advanced Excel-based inventory management and reporting system designed to monitor medicine stock, expiry status, inventory value, and supplier performance. The dashboard provides an interactive view of inventory data, helping users quickly identify low-stock and near-expiry medicines.

---

🎯 Objectives

- Track medicine inventory efficiently
- Monitor stock availability and reorder levels
- Identify expired and near-expiry medicines
- Analyze inventory value by category and supplier
- Build an interactive dashboard for reporting

---

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- XLOOKUP
- IF & COUNTIF
- Conditional Formatting
- Slicers
- Data Validation
